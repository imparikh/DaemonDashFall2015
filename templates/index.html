SmartSpenders

Daemon Dash 2015

Using Capital One's API Nessie, we were able to create a program to successfully monitor children's spending habits. This would ideally be used by parents as either a web or mobile app as a monitoring tool. SmartSpenders provides the unique prediction tool, in which it analyzes how the child is spending his or money, and tells the parent how much money he or she will spend by the end of the month, using existing data.
